# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2020-03-08

### Added

- track completion %, hours logged, title, blocker, and reminder
- editable task fields
- data sync/persistence
- swipe and pull gestures
- date/time picker
- recurrence period picker
- numeric or alphanumeric keyboard depending on field
- full iOS support
- limited android support
